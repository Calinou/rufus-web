Rufus

話都冇咁易 - 就整到隻 USB Boot 機手指 ~

有咗 Rufus，就有得格式化同埋整返隻 USB Boot 機手指，好似 USB 隨身手指、Memory Stick 卡咁 - 得咗。

試過下低咁嘅情形，就知佢幫到你手：

  • 想入隻 ISO 裝機檔落 USB Boot 機手指嗰陣 - 例如 Windows、Linux、UEFI 系統 之類。
  • 想用部冇裝 OS 嘅機嗰陣
  • 想洗洗 BIOS 或者另類 DOS 韌體嗰陣
  • 想行啲底層實用架撐嗰陣

細還細，齊還齊 - Rufus 樣樣都有齊！

仲有，Boot 機邊使九秒九，Rufus 實情快過保特！其實 UNetbootin、Universal USB Installer、Windows 7 USB download tool 佢咃入 Win7 ISO 落手指到已經夠快㗎喇，Rufus 仲要快成倍咁多！用 Rufus 入 Linux ISO 落手指到，就仲快多幾錢重！ (1)
碌落網頁最底，嗰到寫低咗 Rufus 行到邊類型系統嘅 ISO 檔。 (2)

執行所需

Windows 7（或者更加新）；32、64位元唔拘。Down 完唔使裝就用得。

Rufus 作者謝意：「我想借今次機會多謝咁多位有份翻譯嘅義工 - 冇佢哋就冇今日嘅 Rufus。就算係個網頁本身，都係一手一腳譯返嚟，先累積到今日廿靈隻語言咁多。用 Rufus 嗰陣睇得到平時嘅語言，相信都係值得感恩嘅！」

DOS 支援後話：

如果做完隻 DOS Boot 機手指，又冇用到美式鍵盤佈局， Rufus 就會根據系統設置，判斷所在地域先，再揀個啱你嘅輸入佈局。咁嘅情況底下，FreeDOS 就有用過 MS-DOS，因為支援到更加多輸入佈局，可以一試。

ISO 檔支援後話：

由 版號 1.1.0 開始，Rufus 畀你用 ISO 檔 入落 USB 手指到當係 Boot 機手指用。

喺 CD 碟入邊抽個 ISO 檔出嚟其實唔難，不過要做啲手腳。大路嘅免費燒碟程式都做到類似功能，例如 CDBurnerXP 同埋 ImgBurn。

FAQ 解答

問到爛嘅問題，都喺 呢邊 搵得到。

籌錢

成日有人問有冇得捐錢畀我，可惜個答案係 - 冇得。

個主要原因係由於我唔覺得捐錢係幫得到軟件開發界；衰啲講句，咁做只會令到一啲冇能力捐到錢嘅用家有愧於心。

當然，真心好想捐錢支持軟件公益嘅人，可以選擇將錢捐畀一間叫做 Free Software Foundation 嘅團體機構，佢哋絕對係令到 Rufus 賴以成功嘅原因。

點都好，我想借今次機會，同大家誠心講聲，唔該！真係好多謝你哋嘅熱情一路咁撐住呢個程式仔。

最後，請繼續自由自在咁用 Rufus 呢個程式仔，冇必要感到一分內疚。就算冇用錢銀撐我都好，更何況你哋本身冇必要咁做。
